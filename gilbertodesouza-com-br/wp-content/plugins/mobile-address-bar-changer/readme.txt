Mobile Address Bar Changer Plugin is allows you to very easily to add custom color on mobile browser address bar in on Android mobile devices.

== Description ==
Mobile Address Bar Changer Change the color of the address bar of chrome browser on android mobile devices. uses the theme-color meta tag which is used in browsers such as Google Chrome to customize the mobile browser. there is a back end setting to change color on address bar on mobile device. We user theme-color meta tag to change address bar color on mobile.

== Installation ==
1. Upload the mobile-address-bar-changer folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Mobile Address Bar Changer
4. Add the main hex color of your theme i.e. #000000

== Frequently Asked Questions ==
= 1) How to Use ? =
Navigate to Appearance > Mobile Address Bar Changer and choose the color you would like to use.

= 2) Which browsers does work with ? =
Currently works only with the mobile version of Google Chrome. More browsers will be added with time as they adopt new meta tags.
